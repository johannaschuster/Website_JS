Kurzanleitung für Javascript Intl API

JavaScript Intl ist eine Bibliothek für die Internationalisierung von Webanwendungen.

Internationalisierung, auch bekannt als i18n, bezieht sich auf den Prozess der Anpassung von Software an verschiedene Sprachen, Kulturen und Ländern.

JavaScript Intl bietet Entwicklern eine einfache und effektive Möglichkeit Anwendungen für verschiedene Sprachen und Kulturen zu lokalisieren. Das bedeutet, dass eine Anwendung in verschiedenen Sprachen angezeigt werden kann, abhängig von den Präferenzen des Nutzers.

Wozu braucht es diese WEB-API ?

Die JavaScript Intl-API bietet eine Vielzahl von Funktionen und Objekten, die Entwickler verwenden können, um die Internationalisierung von Webanwendungen zu erleichtern.

Das am häufigsten verwendete Objekt ist das Intl.DateTimeFormat-Objekt, das Entwicklern die Möglichkeit gibt, Datums- und Uhrzeitangaben in verschiedenen Formaten anzuzeigen. So zum Beispiel das Anzeigen von Tagen vor Monaten in einigen Ländern oder das Anzeigen von Stunden im 24-Stunden-Format.

Die Funktion DateTimeFormat kann verwendet werden, um ein Datum in das gewünschte Format zu bringen.

Hier ein Beispiel: 


        constdate = newDate('2023-03-21'); 
        constformatter = newIntl.DateTimeFormat('de-DE', { 
          weekday: 'long', 
          year: 'numeric', 
          month: 'long', 
          day: 'numeric', 
        }); 
        console.log(formatter.format(date)); 
        // Ausgabe: "Dienstag, 21. März 2023"

In diesem Beispiel wird das Datum "2023-03-21" für den Wochentag, den Monat und das Jahr im deutschen Sprachraum formatiert. Der DateTimeFormat-Konstruktor akzeptiert ebenfalls verschiedene Optionen, um das Ausgabeformat anzupassen.

Ein weiteres nützliches Objekt in JavaScript Intl ist das Intl.NumberFormat-Objekt, mit dem Entwickler Zahlen in verschiedenen Formaten anzeigen können. Hierzu gehören beispielsweise die Anzeige von Dezimaltrennzeichen, Tausendertrennzeichen und Währungssymbolen.

Hier ein Beispiel:


        constnumber = 1234.567; 
        constformatter = newIntl.NumberFormat('de-DE', { 
          style: 'currency', currency: 'EUR', 
        }); 
        console.log(formatter.format(number)); 
        // Ausgabe: "1.234,57 €"

In diesem Beispiel wird die Zahl 1234.567 in das Währungsformat für Euro im deutschen Sprachraum formatiert. Der NumberFormat-Konstruktor akzeptiert verschiedene Optionen, um das Ausgabeformat anzupassen.

Darüber hinaus bietet JavaScript Intl auch Unterstützung für Sprach-Tagging, eine Methode zur Identifizierung der Sprache, in der Text geschrieben ist. Die Sprache wird dabei durch einen eindeutigen Code identifiziert, der aus einer Sprach- und Regionalkennung besteht.

Hier ein Beispiel:


        constmessages = { 
          en: { 
            greeting: 'Hello, world!', 
          }, 
          de: { greeting: 'Hallo, Welt!', }, 
        }; 
        constlanguage = navigator.language; 
        consttranslatedMessage = messages[language].greeting; 
        console.log(translatedMessage); 
        // Ausgabe: "Hallo, Welt!" (für den Sprachraum "de")

In diesem Beispiel wird die Variable language verwendet, um die bevorzugte Sprache des Nutzers zu ermitteln. Dann wird der entsprechende Text aus dem messages-Objekt abgerufen und angezeigt.
